Satakunta Events API v2

This Linked events API contains data from the Event Calendar of the City of Pori and from other event organizers from Satakunta region.

Satakunta Events is developed by Dyme Solutions Oy and based on an implementation of an open source Linked Events API developed by the City of Helsinki.
